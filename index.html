‚ú® Pan-Diagonal Magic Square: Keajaiban Diagonal ‚ú®

Persegi ajaib pan-diagonal (atau diabolik) adalah persegi ajaib di mana semua diagonal terputus (broken diagonals) juga memiliki jumlah ajaib yang sama.

üéØ Definisi Formal

Untuk persegi $n \times n$ dengan entri $a_{ij}$ ($i,j = 0,\dots,n-1$), disebut pan-diagonal jika untuk semua $k = 0,\dots,n-1$:

1. Diagonal utama terputus:

$$\sum_{i=0}^{n-1} a_{i, (i+k) \bmod n} = M$$

2. Diagonal sekunder terputus:

$$\sum_{i=0}^{n-1} a_{i, (k-i) \bmod n} = M$$
üîß Syarat Eksistensi
  • Orde 3: Tidak mungkin ada persegi ajaib pan-diagonal
  • Orde 4: Mungkin, contoh klasik dari D√ºrer
  • Orde 5: Banyak kemungkinan, termasuk konstruksi linear-modular tertentu
  • Orde $n$: Hanya mungkin untuk $n$ yang tidak habis dibagi 2 atau 3

‚ú® Metode Konstruksi Magic Square ‚ú®

Berbagai metode telah dikembangkan selama berabad-abad untuk membangun persegi ajaib, dari algoritma sederhana hingga teknik matematika kompleks.

üéØ Metode Siamese (De la Loub√®re)

Untuk orde ganjil $n$:

  1. Letakkan 1 di tengah baris teratas
  2. Bergeser ke atas-kanan untuk angka berikutnya
  3. Jika keluar batas, "melipat" ke sisi berlawanan
  4. Jika sel terisi, turun satu sel dari posisi sebelumnya

Algoritma formal:

Input: n ganjil
Output: Persegi ajaib n√ón

1. i = 0, j = (n-1)/2
2. For k = 1 to n¬≤:
   a. A[i][j] = k
   b. new_i = (i-1) mod n
   c. new_j = (j+1) mod n
   d. If A[new_i][new_j] terisi:
        i = (i+1) mod n
      Else:
        i = new_i, j = new_j
üî¢ Metode Linear-Modular

Untuk $n$ ganjil, pilih $a,b$ dengan $\gcd(a,n)=\gcd(b,n)=1$: $$m_{ij} \equiv a \cdot i + b \cdot j + c \pmod{n}$$ Kemudian mapping ke angka 1 sampai $n^2$.

‚öôÔ∏è Metode Grup dan Permutasi

Menggunakan aksi grup $D_4 \rtimes A_n$:

  • $D_4$: Rotasi dan refleksi geometri
  • $A_n$: Permutasi genap pada indeks atau nilai
  • Produk semi-langsung: Kombinasi aksi geometri dan permutasi
üîç Metode Backtracking

Untuk orde besar atau properti khusus:

  1. Isi sel secara rekursif
  2. Verifikasi kondisi di setiap langkah
  3. Backtrack jika tidak mungkin
  4. Optimasi dengan heuristik dan constraint programming
üìê Metode Komposisi

Untuk orde komposit $n = p \times q$:

  • Buat persegi ajaib orde $p$ (template)
  • Buat persegi ajaib orde $q$ (pengisi)
  • Substitusi setiap sel template dengan pengisi yang ditransformasi
  • Hasilnya adalah persegi ajaib orde $n$
